🤖 Gemini 3.x 系統模型機制與連結說明

1. 最新模型狀況：

  • 3.1 Flash-Lite：回覆速度最快，適合簡單的常規性提問。
  • 3.5 Flash：全方位協助模型，為本服務預設的核心 AI。
  • 3.1 Pro：進階數學、程式設計與推論模型，性能最為強大。

2. 思考程度機制 (Thinking Mode)：

  • 標準：常規的生成速度，適用一般提問。
  • 延長：給予較充足的生成處理資源，適用長回答。
  • Deep Think (限 Pro 模型)：啟用平行多推論演算，消耗地端運算額度較高，且系統會對操作全程記錄以供安全合規審計。

3. 地端 Cookies 連線與隔離：

本網關為 Local-First 設計，免申請付費 API Key。連線時，程式會自動從本機的 config/storage_state.json 中提取 Cookies 來維持與官方網頁版的安全連線，並利用獨立 Session ID 進行多用戶會話隔離。
